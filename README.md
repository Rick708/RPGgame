# RPGgame
タップしながらクエストを進行するRPG

## 内容
クエストを進行しながら道中にいるモンスターを倒せ！最後にはお宝があるので獲得するとゲーム終了です。


## 環境環境
unity（２０１９3.6f1）

C#
